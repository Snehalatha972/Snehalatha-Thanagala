వారసత్వ ఉద్యోగాల సాధనపై గుర్తింపు సంఘం టీబీజీకే ఎస్‌ నాయకులు న్యాయ సలహాల పేరుతో కాలయాపన చేస్తూ కార్మికులను తప్పుదోవ పట్టిస్తున్నారని జాతీయ కార్మిక సంఘాలైన సింగరేణి కాలరీస్‌ సింగరేణి కాలరీస్‌ ఎంప్లాయీస్‌ యూనియన్‌ ( సీఐటీయూ ) రాష్ట్ర అధ్యక్షుడు రాజిరెడ్డి, గట్టయ్య ( ఏఐటీయూసీ ) , జనక్‌ప్రసాద్‌ ( ఐఎన్‌టీయూసీ ) , రియాజ్‌అహ్మద్‌ ( హెచ్‌ఎంఎస్‌ ) అన్నారు.
టీజీబీకేఎస్‌ అధ్యక్ష,ప్రధాన కార్యదర్శులు బోగస్‌ మాటలు చెప్పి కార్మికులను మోసం చేస్తే ఖబర్దార్‌ అని హెచ్చరించారు.
వారసత్వ ఉద్యోగాల సాధన కోసం సింగరేణిలో 9రోజులపాటు సమ్మె చేసి నైతికవిజయం సాధించామన్నారు.
కోర్టు తీర్పు వెలువడి నాలుగు నెలలు గడిచినా న్యాయసలహాలు తీసుకుంటామని కాలయాపన చేయడంతో అంతర్యమేంటని ప్రశ్నించారు.
62.35 వేల లక్షల టన్నుల స్టాక్‌ నిల్వలను సమ్మె కాలంలోనే ఉత్పత్తి చేసినట్టు చూపించి టీబీజీకేఎస్‌ నాయకులు కేంద్ర, రాష్ట్ర ప్రభుత్వాలను తప్పుదోవ పట్టించారని విమర్శించారు.
