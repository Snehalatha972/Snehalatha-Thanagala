తెలంగాణ రాష్ట్ర పొల్యూషన్‌ కంట్రోల్‌ బోర్డ్‌ ( టీఎస్‌పీసీబీ ) లోని అనలిస్ట్‌ గ్రేడ్‌-2 పోస్టుల రాతపరీక్ష ఆదివారం ప్రశాంతంగా జరిగింది.
కంప్యూటర్‌ బేస్డ్‌ రాతపరీక్ష ( సీబీఆర్‌టీ ) పద్ధతిలో నిర్వహించిన ఈ పరీక్షకు 20,457 మంది అభ్యర్థులు దరఖాస్తు చేయగా, 10,689 ( 52 శాతం ) మంది హాజరయ్యారు.
ఈ మేరకు టీఎస్‌పీఎస్సీ కార్యదర్శి ఏ వాణీప్రసాద్‌ ఆదివారం ఓ ప్రకటన విడుదల చేశారు.
ఈ పరీక్షలో హైదరాబాద్‌, కరీంనగర్‌, వరంగల్‌ పట్టణాల్లో 62 పరీక్షా కేంద్రాలను ఏర్పాటు చేశామని పేర్కొన్నారు.
హైదరాబాద్‌లో 18,317 మందికి 9,279 మంది, కరీంనగర్‌లో 1,540 మందికి 985 మంది, వరంగల్‌లో 600 మందికి 425 మంది చొప్పున 20,457 మందికి 10,689 మంది అభ్యర్థులు హాజరయ్యారని తెలిపారు.
ఉదయం పది గంటల నుంచి మధ్యాహ్నం 12.30 గంటల వరకు జనరల్‌ స్టడీస్‌ అండ్‌ జనరల్‌ ఎబిలిటీస్‌, మధ్యాహ్నం 2.30 నుంచి సాయంత్రం 5 గంటల వరకు సబ్జెక్టు పేపర్ల రాతపరీక్షలు నిర్వహించామని పేర్కొన్నారు.
టీఎస్‌పీసీబీలోని మిగిలిన పోస్టులకు ఈనెల 13,14 తేదీల్లో రాతపరీక్షలు జరుగనున్నాయి.
