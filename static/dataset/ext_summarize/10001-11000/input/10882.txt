బహుళజాతి కంపెనీల రాకతో చిరువ్యాపారులకు నష్టం వాటిల్లి లక్ష లాది మంది వ్యాపారులు రోడ్డునపడే ప్రమాదం ఉందని సీపీఐ ( ఎం ) రాష్ట్ర కార్యదర్శి తమ్మినేని వీరభద్రం అన్నారు.
హన్మకొండలోని నేరెళ్ల వేణుమాధవ్‌ కళాప్రాంగణంలో తెలంగాణ వ్యాపారస్తుల గోస పేరుతో బహిరంగ సభను వ్యాపారుల జాయింట్‌ యాక్షన్‌ కమిటీ ( తెలంగాణ రాష్ట్రం ) చైర్మెన్‌ పుప్పాల రజనీకాంత్‌ అధ్యక్షతన ఆదివారం నిర్వహించారు.
ఈ సందర్భంగా తమ్మినేని మాట్లాడుతూ గతంలో బహుళజాతి కంపెనీలు ఇండియాలో వ్యాపారం చేయాలంటే అనుమతి తీసుకుని వ్యాపారం చేసేయ న్నారు.
నేడు ఆ నిబంధనను పాలకులు ఎత్తేశారని, దీంతో జపాన్‌, అమెరికా, ఇటలీల్లోని బహుళ జాతి సంస్థలు భారతదేశంలో ఉన్న విస్తారమైన మార్కెట్‌ను కబలించే ప్రయత్నం చేస్తున్నాయని అన్నారు.
ఇందులో భాగంగానే బహుళ జాతికం పెనీలైన వాల్‌ మార్టు, డీ మార్టులు దేశంలో చొరబడి తాపీగా వ్యాపారాలు కొనసాగిస్తున్నాయన్నారు.
లక్షలాది మంది వ్యాపారులే కాకుండా కోట్లాది మంది కార్మికులు, ఉద్యోగులు రోడ్డున పడ్డారన్నారు.
మొదటి నుంచీ సీపీఐ ( ఎం ) బహుళజాతి కంపెనీల చొరబాటు విషయంలో వ్యతిరేకించి నట్టు ఆయన తెలిపారు.
సీపీఐ మాజీ ఎమ్మెల్యే గుండా మల్లేష్‌, కాంగ్రెస్‌ పార్టీ రాష్ట్ర నాయకులు మల్లు రవి మాట్లాడుతూ వాల్‌ మార్టు వల్ల చిరువ్యాపారస్తులు నష్టపోయే అవకాశం ఉందని, దీనికి వ్యతిరేకంగా అందరూ ఐక్యంగా పోరాడాలని పిలుపునిచ్చారు.
