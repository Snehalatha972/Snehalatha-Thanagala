రాష్ట్రంలో డిగ్రీ ప్రథమ సంవత్సరం ఆన్‌లైన్‌ ప్రవేశాల్లో భాగంగా కాలేజీల్లో విద్యార్థులు చేరేందుకు గడువును తెలంగాణ రాష్ట్ర ఉన్నత విద్యామండలి ( టీఎస్‌సీహెచ్‌ఈ ) పొడిగించింది.
కాలేజీల్లో చేరే గడువును ఈనెల 24వ తేదీ వరకు పెంచింది.
షెడ్యూల్‌ ప్రకారం మంగళవారంతో కాలేజీల్లో చేరే గడువు ముగిసింది.
సాంకేతిక సమస్యలు తలెత్తడం, విద్యార్థులకు సీట్ల కేటాయింపు సమాచారం సక్రమంగా అందకపోవడం వంటి ఇబ్బందులు ఉత్పన్నమయ్యాయి.
ఈ సమస్యలపై మంగళవారం ఉన్నత విద్యామండలి కార్యాలయంలో విశ్వవిద్యాలయాల రిజిస్ట్రార్లతో టీఎస్‌సీహెచ్‌ఈ చైర్మెన్‌ తుమ్మల పాపిరెడ్డి, వైస్‌ చైర్మెన్‌ ఎస్‌ మల్లేష్‌, డిగ్రీ ఆన్‌లైన్‌ సర్వీసెస్‌ తెలంగాణ ( దోస్త్‌ ) కన్వీనర్‌ కే వెంకటాచలం సమావేశయ్యారు.
అనంతరం మీడియాతో పాపిరెడ్డి, వెంకటాచలం, మల్లేష్‌ మాట్లాడుతూ విద్యార్థుల ఇబ్బందులను దృష్టిలో ఉంచుకొని కాలేజీల్లో చేరే గడువును ఈనెల 24వ తేదీ వరకు పొడిగిస్తున్నట్టు చెప్పారు.
రెండో విడత కౌన్సెలింగ్‌ తేదీలను త్వరలోనే వెల్లడిస్తామని అన్నారు.
ఈనెల 24వ తేదీ వరకు ఇంటర్‌ అడ్వాన్స్‌డ్‌ సప్లిమెంటరీలో ఉత్తీర్ణులైనవారు, ఇప్పటి వరకు బయోమెట్రిక్‌ ద్వారా నమోదు చేసుకోని విద్యార్థులు మీసేవ, ఈసేవ, హెల్ప్‌లైన్‌ కేంద్రాల్లో తమ పేర్లను నమోదు చేసుకోవాలని కోరారు.
1,088 డిగ్రీ కాలేజీల్లో 1,40,067 మంది విద్యార్థులకు సీట్లు కేటాయించామని వివరించారు.
మంగళవారం నాటికి 90 వేల మంది అలాట్‌మెంట్‌ ఆర్డర్‌ను డౌన్‌లోడ్‌ చేసుకున్నారని చెప్పారు.
రాష్ట్రంలో అనుమతి లేకుండా నిబంధనలకు విరుద్ధంగా నడుస్తున్న మూడు విశ్వవిద్యాలయాలు, ఎనిమది వ్యవసాయ కాలేజీలపై చర్యలు తీసుకోవాలని తెలంగాణ రాష్ట్ర ఉన్నత విద్యామండలి ( టీఎస్‌సీహెచ్‌ఈ ) డిమాండ్‌ చేసింది.
రాష్ట్రంలో యూజీసీ అనుమతుల్లేకుండా, రాష్ట్ర ప్రభుత్వం ఆమోదం లేకుండా మూడు విశ్వవిద్యాలయాలు, ఎనిమిది వ్యవసాయ కాలేజీలు నకిలీగా నడుస్తున్నాయని తెలిపింది.
ఈ మేరకు యూనివర్సిటీ గ్రాంట్స్‌ కమిషన్‌ ( యూజీసీ ) కార్యదర్శికి తెలంగాణ రాష్ట్ర ఉన్నత విద్యామండలి చైర్మన్‌ ప్రొఫెసర్‌ తుమ్మల పాపిరెడ్డి మంగళవారం ఫిర్యాదు చేశారు.
ఉత్తరప్రదేశ్‌లోని ఆగ్రాలో ఉన్న డాక్టర్‌ బిఆర్‌ అంబేద్కర్‌ విశ్వవిద్యాలయం, రాజస్థాన్‌లోని ఝుంఝునులో ఉన్న సింఘానియా విశ్వవిద్యాలయం, ఉత్తరప్రదేశ్‌లోని ఫిరోజాబాద్‌లో ఉన్న జేఎస్‌ విశ్వవిద్యాలయాలు ఎలాంటి అనుమతి లేకుండానే నడుస్తున్నాయని పేర్కొన్నారు.
తెలంగాణలో ఎనిమిది విద్యాసంస్థలూ అనుమతి లేకుండానే నడుస్తున్నాయని తెలిపారు.
