తెలంగాణ ఎంసెట్‌- 2017 ఫలితాల్లో ఏపి రాష్ట్ర విద్యార్థులు సత్తా చాటారు.
ఇంజినీరింగ్‌, అగ్రికల్చర్‌, ఫార్మసీ విభాగాల్లో టాప్‌ 5 ర్యాంకులతో పాటు అత్యుత్తమ ర్యాంకులు సాధించారు.
హైదరాబాద్‌ జెఎన్‌టియు క్యాంపస్‌లో తెలంగాణ ఉన్నత విద్యామండలి చైర్మన్‌ తుమ్మల పాపిరెడ్డి సోమవారం టి ఎంసెట్‌ ఫలితాలను విడుదల చేశారు.
ఇంజినీరింగ్‌ విభాగంలో గుంటూరుకు చెందిన గోరంట్ల జయంత్‌ హర్ష ( 156 మార్కులు ) మొదటి ర్యాంకు సాధించారు.
శ్రీకాకుళం జిల్లాకు చెందిన కిల్లరి రాంప్రసాద్‌ ( 156 మార్కులు ) రెండో ర్యాంకు, భద్రాద్రి కొత్తగూడెంకు చెందిన అవ్వారి భరద్వాజ్‌ ( 155మార్కులు ) మూడో ర్యాంకు, అమలాపురానికి చెందిన దొడ్డి శైలేంద్ర బత్తిబాబు ( 155మార్కులు ) నాలుగో ర్యాంకు, హైదరాబాద్‌కు చెందిన వబిలిసెట్టి మొహన్‌ అభ్యాస్‌ ( 155మార్కులు ) ఐదో ర్యాంకు సాధించారు.
తాడిపత్రి విద్యార్ధి చగం దిలీప్‌కుమార్‌ రెడ్డి, మచిలీపట్నం విద్యార్ధి అబ్దుల్‌ మొయిజ్‌ ఆరు, తొమ్మిది ర్యాంకులు సొంతం చేసుకున్నారు.
