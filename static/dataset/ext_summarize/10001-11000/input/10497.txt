కోల్‌మైన్‌ ప్రావిడెంట్‌ ఫండ్‌ ( సీఎం పీఎఫ్‌ ) ను ఎంప్లాయి ప్రావిడెంట్‌ ఫండ్‌ ( ఇపీఎఫ్‌ ) లో విలీనం చేసే ప్రతిపాదనను కేంద్ర ప్రభుత్వం విరమించుకుంది.
కార్మికులకు తీవ్ర నష్టం కలిగించేందుకు కేంద్ర ప్రభుత్వం చేసిన కుట్రను తీవ్రంగా వ్యతిరేకిస్తూ ఈనెల 19 నుంచి 21వ తేదీ వరకు దేశవ్యాప్త సమ్మెకు జాతీయ కార్మిక సంఘాలు పిలుపు నిచ్చింది.
దీంతో కేంద్ర ప్రభుత్వం దిగొచ్చి ఇపీఎఫ్‌ లో విలీనం ప్రతిపాదనను విరమించు కుందని సింగరేణి కాలరీస్‌ ఎంప్లాయీస్‌ యూనియన్‌ ( సీఐటీయూ ) రాష్ట్ర ప్రధాన కార్యదర్శి మందా నర్సింహాssరావు తెలిపారు.
కేంద్రం డీవైసీఎల్‌సీ ఎదుట ఆదివారం అలాంటి ప్రతిపాదనను విరమించు కున్నట్టు ఒప్పందం చేసుకోవడంతో సోమవారం నుంచి నిర్వహించే సమ్మెను మూడు నెలలు వాయిదా వేసుకున్నట్టు చెప్పారు.
ఇది జాతీయ కార్మిక సంఘాల ఘన విజయమని, కార్మికుల ఐక్యతకు నిదర్శమని అన్నారు.
సీఎంపీఎఫ్‌ ఫండ్‌లో జరిగే నష్టాన్ని, పెన్షన్‌లో జరిగే దుష్ఫలితాన్ని కార్మికులు తీవ్రంగా వ్యతిరేకించడం వల్ల కేంద్రం నిర్ణయాన్ని ఉపసంహరించుకుందన్నారు.
రెండు నెలల్లో వేతన ఒప్పందాన్ని పరిష్కరించాలన్న జాతీయ కార్మిక సంఘాల డిమాండ్‌ను కేంద్రం అంగీకరించినట్టు తెలిపారు.
ఇదే ఒరవడితో సింగరేణిలో వారసత్వ ఉద్యోగాల సాధన కోసం నిర్వహిస్తున్న సమ్మెను కూడా విజయవంతం చేయాలని పిలుపునిచ్చారు.
