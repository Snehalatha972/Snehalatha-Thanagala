ప్రింట్‌, ఎలక్ట్రానిక్‌ మాధ్యమాలో విలేకర్లుగానూ, వార్తపఠితలుగానూ పనిచేస్తున్న వారికి లలిత కళాస్రవంతి శ్రీ త్యాగరాయ గానసభలో ఆదివారం కన్నుల పండుగగా సత్కరించి వారిపట్ల అభిమానాన్ని ప్రకటించింది.
ముఖ్య అతిథిగా హాజరైన తెలంగాణ టెలికామ్‌ జనరల్‌ మేనేజర్‌ రాంచందర్‌ మాట్లాడుతూ పెద్దలు తమ పిల్లలకు చిన్నతనంలో నృత్య, సంగీతంలో శిక్షణ ఇప్పిస్తారని, వారు పెద్దైన తరువాత ఆ కళలను వదిలివేస్తే తల్లిదండ్రులకు అసంతృప్తి మిగులుతుందన్నారు.
మాధ్యమాల్లో తమ వృత్తి ధర్మాన్ని నిర్వర్తిస్తున్న వారిని సత్కరించడం ముదావహం అన్నారు.
వేదికపై సాంస్కృతిక సౌజన్య దాత జీఎస్‌ శ్రీనివాస్‌, పారిశ్రామిక వేత్తలు లక్ష్మారెడ్డి, భద్రిప్రసాద్‌ పాల్గొనగా, సంస్థ అధ్యక్షులు జి.దీక్షితులు స్వాగతం పలికారు.
పురస్కారాలను అందుకున్న వారిలో టీవీ యాంకర్స్‌ సుమతి ( టీవీ 9 ) , ప్రతిభ ( ఎన్టీవీ ) , ఉదయశ్రీ ( టీవీ 5 ) , గౌతమి ( సాక్షి ) , జయ ( టెన్‌టీవీ ) , రాజేశ్వరీ ( ఏబిఎన్‌ ఆంధ్రజ్యోతి ) , తదితరులతో పాటు వార్తాపఠితలు తరుణ ( టీవీ 9 ) , ఉమ ( టీవీ 5 ) , శ్రీదేవీ ( సాక్షి ) , రాధిక ( టెన్‌టీవీ ) , వీణావాణి ( జెమిని న్యూస్‌ ) , సరోజ ( ఏబిఎన్‌ ఆంధ్రజ్యోతి ) , కృష్ణవేణి ( భారత్‌ టుడే ) లకు ఈవి రాజయ్య స్మారక పురస్కారాలు బహుకరించారు.
కార్యక్రమానికి తొలుత రమాదేవి శిష్యబృందంచే భరతనాట్య ప్రదర్శన, వి.లక్ష్మి శిష్యబృందంచే అన్నమయ్య సంకీర్తన గానగోషి జరిగింది.
