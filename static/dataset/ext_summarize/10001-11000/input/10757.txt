రాష్ట్రంలో అసోసియేట్‌ ప్రొఫెసర్లుగా పనిచేస్తున్న 12 మంది అనస్ఖీషియా డాక్టర్లకు ప్రభుత్వం ప్రొఫెసర్లుగా పదోన్నతి కల్పించింది.
ఈ మేరకు వారికి పోస్టింగ్‌ కూడా కేటాయిస్తూ రాష్ట్ర వైద్యఆరోగ్యశాఖ ముఖ్యకార్యదర్శి రాజేశ్వర్‌ తివారీ బుధవారం ఉత్తర్వులు జారీ చేశారు.
డాక్టర్‌ సాధనరారు ప్రస్తుతం గాంధీ మెడికల్‌ కాలేజీలో అసోసియేట్‌ ప్రొఫెసర్‌ ( అ.ప్రొ. ) పనిచేస్తున్నారు.
పదోన్నతిపై ఇక నుంచి నీలోఫర్‌ ఆస్పత్రిలో విధులు నిర్వహించనున్నారు.
అదేవిధంగా డాక్టర్‌ కే.పావని ( అసోసియేట్‌ ప్రొఫెసర్‌ - ఉస్మానియా మెడికల్‌ కాలేజీ ) ప్రొఫెసర్‌గా ఉస్మానియా జనరల్‌ ఆస్పత్రిలో, డాక్టర్‌ హెఎల్‌ బేబీరాణి ( అ.ప్రొ.- గాంధీ మెడికల్‌ కాలేజీ ) ప్రొఫెసర్‌గా గాంధీ మెడికల్‌ కాలేజీలో, డాక్టర్‌ రాజేంద్ర ( అ.ప్రొ.- నిజామాబాద్‌ ప్రభుత్వ మెడికల్‌ కాలేజీ ) ప్రొఫెసర్‌గా ఓజీహెచ్‌లో, డాక్టర్‌ ఎం.వీరేషం ( అ.ప్రొ.- గాంధీ మెడికల్‌ కాలేజీ ) ప్రొఫెసర్‌గా జీఎంసీలో, డాక్టర్‌ ఏ అభిమన్యు సింగ్‌ ( అ.ప్రొ.- నిజామాబాద్‌ ప్రభుత్వ మెడికల్‌ కాలేజీ ) ప్రొఫెసర్‌గా చెస్ట్‌ ఆస్పత్రిలో, డాక్టర్‌ ప్రవీణ్‌కుమార్‌ ( అ.సో.-వరంగల్‌ కాకతీయ మెడికల్‌ కాలేజీ ) ప్రొఫెసర్‌గా మహాత్మాగాంధీ మెమోరియల్‌ ఆస్పత్రి, డాక్టర్‌ పీవీ శివ ( అ.సో.- నిజామాబాద్‌ ప్రభుత్వ మెడికల్‌ కాలేజీ ) ప్రొఫెసర్‌గా నీలోఫర్‌ ఆస్పత్రిలో, డాక్టర్‌ కే.ఉమ ( అ.సో.- ఓఎంసీ ) ప్రొఫెసర్‌గా ఈఎన్‌టీ ఆస్పత్రి, డాక్టర్‌ నాగార్జున రెడ్డి ( అ.సో.- హన్మకొండ మెటర్నిటీ ఆస్పత్రి ) ప్రొఫెసర్‌గా హన్మకొండ మెటర్నిటీ ఆస్పత్రి, డాక్టర్‌ సురేష్‌ చందర్‌ ( అ.ప్రొ - జీఎంసీ ) ప్రొఫెసర్‌గా జీఎంసీలో, డాక్టర్‌   రూపా కుమారి ( అ.ప్రొ.- నిజామాబాద్‌ ప్రభుత్వ మెడికల్‌ కాలేజీ ) ప్రొఫెసర్‌గా సరోజిని దేవీ కంటి ఆస్పత్రి బదిలీ అయ్యారు.
కాగా, వీరికి ప్రమోషన్లు కల్పించడంపై తెలంగాణ ప్రభుత్వ వైద్యుల సంఘం రాష్ట్ర అధ్యక్ష, ప్రధాన కార్యదర్శులు డాక్టర్‌ పుట్ల శ్రీనివాస్‌, డాక్టర్‌ రవిశంకర్‌ ముఖ్యమంత్రి కేసీఆర్‌కు, వైద్యశాఖ మంత్రి డాక్టర్‌ లక్ష్మారెడ్డికి ధన్యవాదాలు తెలిపారు.
