కేంద్ర, రాష్ట్ర ప్రభుత్వాలు ప్రజా ఆరోగ్యంపై అనుసరిస్తున్న విధానాలు, వాటి పర్యవసానాలు, పరిష్కార మార్గాలపై చర్చించాల్సిన అవసరముందని ప్రజారోగ్య వేదిక ( పిఎవి ) రాష్ట్ర కన్వీనర్‌ డాక్టర్‌ సురేష్‌ అన్నారు.
విశాఖలోని పౌర గ్రంథాలయంలో రిటైర్డ్‌ డిఎంహెచ్‌ఒ పి.రామారావు అధ్యక్షతన ఆదివారం ఏర్పాటుచేసిన సమావేశంలో మార్చి 4న నిర్వహించే జాతీయ సదస్సు నిర్వహణ ఏర్పాట్లు, పర్యవేక్షణపై చర్చించారు.
అందుకోసం ఒక కమిటీని ప్రకటించారు.
ఈ సందర్భంగా సురేష్‌ ప్రసంగిస్తూ, ప్రజా ఆరోగ్య విధానాలపై విసృతంగా చర్చ జరగాలనే ఉద్దేశంతో 2015 అక్టోబర్‌లో ప్రజా ఆరోగ్య వేదిక ఏర్పాటు చేసినట్లు తెలిపారు.
ఆరోగ్య శ్రీ వంటి పథకాలు, తాజాగా కేంద్రం ప్రకటించిన ఆయుష్మాన్‌భవ పథకాల వల్ల తలెత్తే దుష్ఫరిణామాలు ప్రభుత్వాలకు తెలిసినా వాటిని తెరపైకి తెస్తున్నాయని చెప్పారు.
ప్రజలకు అందుబాటులో ఉండే క్లినిక్‌లు, నర్సింగ్‌ హోమ్‌లు, ప్రాక్టీషనర్లను దెబ్బతీసి మొత్తం రోగులను కార్పొరేట్‌ ఆసుపత్రుల వైపు నడిపించి ప్రజలను వైద్య బీమాపై ఆధారపడేలా చేస్తున్నాయన్నారు.
ప్రభుత్వ ఆసుపత్రులను నిర్వీర్యం చేస్తున్నాయని తెలిపారు.
