ఆల్‌ ఇండియా ఇన్స్‌స్ట్యూట్‌ ఆఫ్‌ మెడికల్‌ సైన్సెస్‌ ( ఎయిమ్స్‌ ) ను రాష్ట్రంలో ఏర్పాటుకు చేయడానికి కేంద్ర ప్రభుత్వం ఆమోదం తెలిపిందని కేంద్ర కార్మిక, ఉపాధి కల్పన శాఖ మంత్రి బండారు దత్తాత్రేయ తెలిపారు.
అదేవిధంగా విద్య, ఆరోగ్య రంగాలకు కేంద్ర ప్రభుత్వం రూ.1200 కోట్లు మంజూరు చేసిందన్నారు.
వరంగల్‌లోని కేఎంసీకి రూ.74 కోట్లు, ఆదిలాబాద్‌ జిల్లాల్లోని ఆర్‌ఐఎమ్‌ఎస్‌కు రూ.72 కోట్లు మంజూరు చేసిందని చెప్పారు.
శుక్రవారం హైదరాబాద్‌లో కేంద్ర ప్రభుత్వ ఆరోగ్య పథకం ( సీజీఎహెచ్‌ఎస్‌ ) వెల్‌నెస్‌ సెంటర్‌ పని తీరును ఆయన పరిశీలించారు.
ఈ సందర్భంగా దత్తాత్రేయ మాట్లాడుతూ విద్య, ఆరోగ్య రంగాలకు కేంద్రం అత్యంత ప్రాధాన్యం ఇస్తున్నదని చెప్పారు.
దేశంలో వైద్యుల కొరతను అధిగ మించేందుకు ప్రభుత్వ వైద్యుల ఉద్యోగ విరమణ కాలాన్ని మూడేండ్లు పెంచారన్నారు.
దేశంలోని అన్ని జిల్లాల్లో డయాలసిస్‌ సెంటర్లను ఏర్పాటు చేయడానికి కేంద్ర ప్రభుత్వం ఆర్థిక సహయాన్ని అందిస్తుందని తెలిపారు.
సీజీహెచ్‌ఎస్‌లను త్వరలోనే పాలి క్లినిక్‌లుగా మార్చనున్నట్టు తెలిపారు.
